## 補足

- 同じ帳票内でも、左側と右側で同名項目が別セルに存在する。
- 一部の項目は1セルではなく、見出しの下や数行下に値が置かれている。
  - 例: `発注者住所` は `G25` ではなく `G27`
  - 例: `工期終了` は `H29` ではなく `H30`
- `監理技術者名 / 主任技術者名` のように、配属区分と氏名が別セルに分かれている項目がある。
- `担当工事内容` や `専門技術者名` はサンプルでは値が入っていない欄があり、未確定のまま。
